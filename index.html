Featured project

Signbot (McHacks 2017 Winner)

Signbot is a tool that translates voice to sign language, displayed using a set of prosthetic arms. It is a multipurpose sign language device including uses such as: a teaching model for new students, a voice to sign translator for live events, or simply a communication device between voice and sign.


Built by Alex Foley, Colin Daly, Clive Chan, and Wilson Wu, McHacks 2017
